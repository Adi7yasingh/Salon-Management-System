How to run the Salon Management System Using PHP and MySQL


1.Extract the zip file and copy msms folder
2.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
3.Open PHPMyAdmin (http://localhost/phpmyadmin)
4.Create a database with name msmsdb
5.Import msmsdb.sql file(given inside the zip package in SQL file folder)
6.Run the script http://localhost/msms
